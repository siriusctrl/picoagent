Control background tools and subagents. Use status for an immediate snapshot, wait for one bounded wait interval, inspect to read a subagent's recent native message sequence, steer to queue a non-interrupting instruction for a running subagent, or stop to cancel one task. Inspect returns the latest 6 messages by default; when has_earlier=true, pass next_before_seq as before_seq to page backward. Steering is inserted after the child's current assistant response and complete tool-call batch, before its next model request. Wait never stops work. Resolve every live task before giving a final answer; terminal results are also delivered as durable background-result messages.
