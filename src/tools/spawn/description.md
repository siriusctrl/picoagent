Start one allowed tool call or general-task subagent in the background and return immediately. For kind=tool, the schema's tool enum is the exact allowed set. The result is JSON with task_id, kind, name, status, and a lifecycle message. Use this only for independent work; wait before consuming a dependency. Direct calls begin in the foreground and may enter the same task lifecycle after their foreground window. Completed results are appended at the next model boundary.
